## Oracle Database
**Multimodelo:** A diferencia de otras bases de datos que solo guardan tablas, Oracle puede guardar documentos JSON, grafos, datos espaciales (coordenadas) y vectores (muy usado ahora para IA).

**Autonomous Database:** Es su versión más moderna. Utiliza Machine Learning para auto-optimizarse. Es decir, la base de datos "aprende" qué consultas son lentas y crea índices automáticamente para acelerarlas.

## Oracle Cloud Infrastructure

**Generación 2 (Gen 2 Cloud):** Oracle diseñó su nube después que Amazon (AWS) y Google, lo que les permitió evitar errores del pasado. Su red es más rápida y aislada, lo que evita que el tráfico de un cliente afecte a otro.

**Servicios Gratuitos (Always Free):** Para tu investigación, te sirve saber que Oracle ofrece una capa gratuita generosa para estudiantes

**Cómputo y GPU:** Tienen alianzas fuertes con NVIDIA para ofrecer potencia de procesamiento brutal, ideal para los modelos de **Análisis de Datos**


## Middleware
**Gestión de APIs:** Es donde se crean y aseguran las rutas (puntos de acceso) para que tus programas de **FastAPI** o apps móviles consuman los datos de forma segura.

**Integración:** Permite que sistemas viejos hablen con sistemas modernos sin tener que programar todo desde cero.
