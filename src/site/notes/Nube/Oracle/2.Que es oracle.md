Oracle ofrece varios serivcios en la nuebe pero el mas importante es el negocio de las bases de datos, sin embargo estos son los servicios que ofrecen en general:
## Base de datos
- **RDBMS:** Un Sistema de Gestión de Bases de Datos Relacionales.

## Oracle Cloud Infrastructure (OCI)
Es su plataforma en la nube, competencia directa de AWS, Azure y Google Cloud. Ofrecen:
- **Servidores y Almacenamiento:** Para alojar aplicaciones de **React** o **FastAPI** como las que tú desarrollas.
- **IA y Machine Learning:** Herramientas para entrenar modelos sin tener que configurar toda la infraestructura desde cero.
- **Exadata:** Son servidores físicos diseñados específicamente para correr bases de datos

## Aplicaciones Empresariales (ERP / CRM)
Muchas de las empresas más grandes del mundo usan software de Oracle para gestionar todo su negocio:
- **NetSuite / Oracle Fusion:** Software para llevar la contabilidad, los recursos humanos y la logística de una empresa.

## Middleware
Es el software que "conecta" diferentes aplicaciones. Por ejemplo, si tienes una app en **React** y necesitas que hable de forma segura con un sistema bancario antiguo, Oracle tiene herramientas (como WebLogic) que hacen ese puente.
