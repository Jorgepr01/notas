## Oracle Cloud "Always Free"
**Lo que obtienes gratis para siempre:** * **2 Bases de Datos Autónomas** (perfectas para tus prácticas pero limitadas para casos reales).
- **Instancias de Cómputo (Máquinas Virtuales):** Incluyendo procesadores ARM Ampere con hasta 24 GB de RAM
- **Almacenamiento de Objetos:** Para guardar imágenes o archivos de tus apps.


### Lo basico en el OCI es:
- **Compute (Cómputo):** Aquí es donde creas tus servidores virtuales (instancias).
- **Oracle Database (Bases de Datos):** Aquí es donde la magia de lo que viste en la web de pruebas se vuelve real y escalable.
	- **Autonomous Database:** Es la opción recomendada. Al crearla, el sistema se encarga de los parches, la seguridad y las copias de seguridad. Puedes elegir entre "Transaction Processing" (para apps como las de React) o "Data Warehouse" (para **Análisis de Datos** pesado).
- **Networking (Redes Virtuales - VCN):** Este es el punto más técnico pero vital. Antes de que nada funcione, debes configurar tu **VCN (Virtual Cloud Network)**.
